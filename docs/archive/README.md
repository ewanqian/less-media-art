## 如何提交案例

1. **选择模板**: 根据你的实践类型选择对应模板
2. **填写内容**: 按照模板提示完成所有部分
3. **删除提示**: 删除所有方括号中的提示文字
4. **提交PR**: 将文件放入对应类型目录，提交Pull Request

## 案例命名规范

```
[作者名]-[主题关键词]-[日期].md

示例:
- ewanqian-meaning-of-creation-20250202.md
- anonymous-constraint-poetry-20250201.md
```

## 案例质量标准

一个高质量的案例应该：

- ✅ 完整记录过程，不只是结果
- ✅ 包含真实的反思，不只是描述
- ✅ 诚实地记录失败和犹豫
- ✅ 与理论概念建立联系
- ✅ 提出开放问题供他人思考

## 隐私与伦理

- 你可以使用化名或匿名提交
- 确保对话内容不包含敏感个人信息
- 如果涉及他人，请获得同意
