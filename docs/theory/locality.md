# 本地性 (Locality)

> "云是别人的电脑。"

## 核心定义

**本地性**是指坚持在个人设备上进行创作运算和数据存储的立场。在少媒体艺术的语境下，这是对云计算和中心化服务的一种主动抵抗。

## 为什么选择本地？

### 1. 数据主权
- 你的数据属于你
- 不会被用于训练商业模型
- 不会被审查或删除

### 2. 物质可见
- 感受硬件的限制
- 感知计算的成本
- 理解技术的物性

### 3. 去中心化
- 不依赖科技巨头
- 抵抗平台垄断
- 维护数字独立性

## 本地性的实践

### 硬件层面
- 使用个人电脑而非云端服务器
- 选择可审计的开源硬件

### 软件层面
- 使用本地运行的AI模型（如Ollama）
- 选择开源可审计的软件

### 数据层面
- 本地存储创作档案
- 定期备份到个人控制的存储介质

## 批评与回应

**批评**: 本地性是否只是技术精英主义？普通用户没有资源搭建本地环境。

**回应**: 本地性不是排斥，而是**选择的可能**。关键是意识到有选择，并在可能的情况下选择本地。
