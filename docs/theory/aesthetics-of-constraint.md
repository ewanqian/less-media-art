# 约束美学 (Aesthetics of Constraint)

> "自由不是选择的多样性，而是选择的必然性。"

## 核心定义

**约束美学**是指通过主动施加限制来激发和塑造创造力的美学理论。在少媒体艺术的语境下，约束不是创作的障碍，而是**创作的条件**。

## 约束的类型

### 形式约束
- 字数限制
- 格式要求
- 媒介选择

### 过程约束
- 时间限制
- 工具限制
- 环境限制

### 概念约束
- 主题限制
- 语义限制
- 逻辑限制

## 约束与创造力的关系

约束通过以下机制激发创造力：

1. **聚焦注意力** — 减少选择，深化探索
2. **强制创新** — 在限制中寻找新路径
3. **显形决策** — 每个选择都变得可见
