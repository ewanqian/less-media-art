# 少媒体，不是一种风格，而是一种质问。

在生成式AI以生产力和奇观席卷一切的当下，我们选择追问：当创作变得无限便捷时，我们失去了什么？

这是一个开源的研究框架，邀请你进行一场反向实践：通过主动地、有意识地施加限制，来重新触摸创作的核心。

---

## 核心立场

**少媒体艺术**并非一个宣言式的"学派"，也非一种固定的视觉风格。它是一个**开放的、持续自我质疑的研究框架**，源于对当下数字艺术生产范式的深度关切。

其核心在于：在生成式人工智能以"无限可能性"和"即时生产力"塑造主流艺术话语的今天，我们提出一种以[媒介减法](theory/media-subtraction.md)、[技术内省](theory/technological-introspection.md)与[过程显形](theory/process-visibilization.md)为核心的批判性实践。

## 理论基底

### 观念艺术的遗产

我们继承并转换了索尔·勒维特"观念先于形式"的命题。在少媒体的语境下，"观念"体现在对**创作条件本身的先决性设置**上——为何选择此种媒介？为何施加此项限制？对这些问题的审慎回答，构成了作品不可见但至关重要的观念内核。

### 对"技术黑箱"的批判

借鉴了弗里德里希·基特勒的媒介物质性思想和当下对算法文化的批判，少媒体艺术强调将技术过程从"黑箱"中拉回前台。**使用本地、轻量、可审计的模型**，正是为了恢复对创作工具的物质性与政治性的认知，对抗云服务与大型模型带来的同质化效率与认知遮蔽。

### "限制作为生成器"的美学传统

从文学领域的"乌力波"（Oulipo）到音乐领域的简约主义，主动设置的规则与限制长久以来被视为激发创造力的引擎。少媒体艺术将这一传统置于数字AI创作的语境中，探究在"智能"工具面前，**人为约束如何成为保持作者主体性与批判意识的必要手段**。

### 后网络时代的"物性"回归

在数字图像近乎无限复制与传播的当下，少媒体艺术通过强调**本地计算的物质性**（硬件、电量、时间消耗）、**过程文件的唯一性**（特定环境生成的日志），以及**交互的笨拙与阻力**，试图重新为数字艺术注入一种可感知的"物性"与"事件性"。

## 如何参与

我们邀请您以**共同研究者**的身份参与，而非信徒或学生。参与路径是多元的：

1. **实践与提交**：运用或改造"方法工具箱"中的思路，完成一次个人的实践，并按照规范将完整的"案例"提交至档案库。
2. **批判与阐释**：对档案库中的现有案例进行深入的批注、批评，或撰写比较分析文章，丰富其理论维度。
3. **扩展与质疑**：为"理论词条"补充新的参考文献或提出修正意见；在"讨论场"发起新的议题辩论。
4. **工具构建**：为落实某些方法，开发新的、更极简的软件工具或脚本，并附上使用哲学说明。

## 结语

"少媒体艺术"在此框架下，不指向某种最终可被辨识的视觉样式。它指向的是一种**持续的、集体的反思路径**——在技术日益决定创作形态的时代，我们如何通过主动的限制，夺回定义的权力？我们如何通过暴露过程，来理解我们与工具之间真正的协作与对抗关系？

这个网站本身，连同其内部不断生长的词条、方法、案例与辩论，便是这一反思路径的具象化。它是一个**未完成的、始终处于构建中的思想模型**，其意义在于提出真问题，并提供一个严谨的场域，让大家一起用实践去探索可能的回答。

欢迎您以严谨的实践或思辨，成为这个研究网络的构建者之一。
