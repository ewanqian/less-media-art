# 讨论与议题

这里是少媒体艺术框架的开放讨论区。任何关于理论、方法、案例的质疑、补充或辩论都可以在这里进行。

## 如何进行讨论

1. **创建新议题**：在GitHub Issues中创建新讨论，标签选择 `discussion`
2. **引用相关内容**：使用链接引用理论词条、案例或方法
3. **保持尊重**：批判观点，而非个人
4. **记录结论**：讨论有结果后，更新相关文档
